 # ☁️ SkyScale Serverless API (AWS Lambda + API Gateway)
A serverless API built using AWS Lambda and API Gateway as part of an MCA Cloud Computing project. It demonstrates backend logic with dynamic query parameters and cloud-based deployment architecture.
## 📌 Project Overview
This project is a serverless backend API developed using AWS Lambda and Amazon API Gateway. It demonstrates cloud-native application development by eliminating server management and using event-driven architecture.
